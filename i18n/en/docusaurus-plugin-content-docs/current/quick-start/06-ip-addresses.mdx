# Polaris IP Addresses

Once configured under Polaris, some webpages may have an error page (Error 504) when
attempting access. In order to resolve this issue, the you must add the following IP addresses to
your **domain Origin Server's** whitelists.

*Note: These are separate whitelisting functions from the Polaris WAAP whitelisting in Access Control.*

## IPv4 list:
- 13.251.210.79
- 18.142.251.243
- 3.1.158.204
- 171.244.28.79
- 210.245.38.74
- 118.69.83.88
- 171.244.128.15
