# Reports

Polaris allows you to download monthly reports containing a summary of the threats faced by
the site that month. The report can be customized to only include certain sections of your
choice.

This article shows how you can download the report and the different sections you may choose
to include in it.

## Downloading Reports

![Download Reports](/img/security-center/overview/report-download.png)

1. Select the sections you would like to include in the report - Bandwidth, Traffic, Attack
Summary, DDoS, Security Event, Incident, and Threat Intelligence.

2. Select the month.

3.  Select the domains you would want in the report - selecting multiple domains will include
them in the same document so if you want the domains to be separated in different documents, export separately

The document will open in a new tab for viewing, which can then be downloaded in PDF format.
