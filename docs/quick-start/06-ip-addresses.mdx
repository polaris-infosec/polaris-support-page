# Địa chỉ IP của Polaris

Sau khi được cấu hình trong Polaris, một số trang web có thể hiển thị trang lỗi (Lỗi 504) khi cố gắng truy cập.
Để giải quyết vấn đề này, bạn phải thêm các địa chỉ IP sau vào danh sách trắng của **máy chủ nguồn của miền** của bạn.

*Lưu ý: Đây là các chức năng danh sách trắng riêng biệt so với danh sách trắng Polaris WAAP trong Access control.*
