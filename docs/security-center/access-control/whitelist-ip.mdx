# Đưa địa chỉ IP vào Whitelist

## Whitelist là gì?

Có hai cách để giới hạn truy cập đến trang web của bạn, cho phép tất cả các lưu lượng truy cập trừ các địa chỉ IP được
chỉ định cụ thể ([Blacklist IP](blacklist-ip)) hoặc chỉ cho phép các địa chỉ IP được chỉ định (Whitelist IP).

## Làm thế nào để cho phép một địa chỉ IP vào Whitelist?

Cài đặt Whitelist IP Address có thể được tìm thấy trong tab `Access control` của cài đặt trang web của bạn.

Để truy cập vào `Settings`, hãy nhấp vào phần menu dropdown trên trang web đã chọn:

![Setting Tab](/img/security-center/overview/settings-tab.png)

Dưới `SITES`, hãy chọn tên miền của bạn và nhấp vào `Settings`.

Tìm tab `Access control` trong Cài đặt:

![Whitelist Ip Tab](/img/security-center/access-control/whitelist-ip-whitelist-ip-tab.png)

Chọn `Whitelist IP Address` trong `Access Control Tab` ở phần cài đặt. Chỉ cần nhập địa chỉ IP mà bạn muốn cho
phép truy cập trong định dạng xxx.xxx.xxx.xxx và nhấp vào tùy chọn `Whitelist`.

## Gỡ bỏ các trang web đã được đưa vào whitelist.

Bạn cũng có thể xóa các trang web đã được phép truy cập từ trang này:

![Remove Whitelist Ip](/img/security-center/access-control/whitelist-ip-remove-whitelist.png)

Đơn giản chỉ cần chọn các địa chỉ IP mà bạn muốn xóa khỏi whitelist và nhấp vào tùy chọn xóa (như được hiển thị ở trên).
